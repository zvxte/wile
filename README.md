# Wile

**Solve chess puzzles from your own games**.

## Setup

Create virtual environment, then install dependencies.
On Linux:
```bash
cd wile
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
See [example](./src/example.py).
